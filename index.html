◈ NEW — Bitcoin Research

Satoshi's Donation Protocol (SDP) — A consensus-layer mechanism that restricts spendability of quantum-vulnerable P2PK outputs (1,720,270 BTC across 45,257 UTXOs with public keys exposed on-chain since 2009–2012). Activates only after block subsidy drops below 1 BTC (halving 6, ~2032). If no cryptographic break occurs, no additional value enters circulation.

  • Scope: Only P2PK with exposed public keys. P2PKH, P2SH, Taproot, and PQC holders are never affected.
  • Key-rotation window: Any living P2PK holder can opt out with one transaction before block ~1,035,000 (~Jan 3, 2028). Ownership preserved forever.
  • Economic trigger: Donation does not activate until GetBlockSubsidy() < 1 BTC (halving 6, ~2032). Before that, nothing changes.
  • Dormant unless needed: If quantum never arrives, the pool never activates. Contingency, not prediction.
  • 21M cap preserved. MAX_MONEY unchanged. Zero new coins.
